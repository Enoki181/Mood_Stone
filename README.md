# Mood Stone

## サービス概要
今のあなたの気分を入力し、あなたの気分に合う「石」をみつけるミニアプリです。  
自分の気分を数値化し、「石」という形を与えるという作業を通して、自分の心を向き合うきっかけを提供できればと思います。  
[https://mood-stone.onrender.com/](https://mood-stone.onrender.com/)

## 使い方
今のあなたの気分を、スライダーで入力してみましょう。  
スライダーを動かすと、スライダーの上の数字が変化します。  
数字を見ながら「今は、このぐらいの気分だな」と感じたら、「石をみつける」ボタンを押しましょう。  
あなたの気分に合う「石」がみつかります。

## 開発めも
- 主に、学習内容の振り返りと、経験のために作成しました。
- 気分の数値に対応する「形」としては、「石」を採用しています。
- おもに色を重視して数値と石の対応を考えているので、石言葉などを考えるとそぐわないと感じられるかもしれません。
- パワーストーンを勧める意図はないので、結果画面の解説はあくまで石についてのものに終始しています。
### 気分の数値に対応する「形」を「石」にした理由
- 今回登録している「石」は、10種類すべて同じ鉱物(quartz)です。鉱物は、含んだ微量元素や紫外線照射などによって色が変化します。このように、もともともっている特徴(微量元素など)と外からの要因(紫外線照射)によって色(気分、気持ち)が変わるのは、心と似ているのでは、と考えたことが「石」にしようと考えたきっかけです。
- 私が鉱物好きで、このようなアプリがほしかったというのももちろん理由のひとつです。
