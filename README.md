# MoodStone

## サービス概要
今のあなたの気分を入力すると、あなたの気分に合う「石」がみつかるアプリです。
自分の気分を数値化し、「石」という形を与えることで、自分の心を向き合うきっかけを提供できればと思います。

## 使い方
今のあなたの気分を、スライダーで入力してみましょう。
スライダーを動かすと、スライダーの上の数字が変化します。
数字を見ながら「今は、このぐらいの気分だな」と感じたら、「石をみつける」ボタンを押しましょう。
あなたの気分に合う「石」がみつかります。

## 開発めも
- 主に、学習内容の振り返りと、経験のために作成しました。
- 気分の数値に対応する「形」としては、「石」を採用しています。
--おもに色を重視して数値と石の対応を考えているので、石言葉などを考えるとそぐわないと感じられるかもしれません。
- 気分の数値に対応する形を「石」にした理由。
-- 私が鉱物好きで、このようなアプリがほしかったからです。
-- 今回登録している「石」は、10種類すべて同じ鉱物(quartz)です。鉱物は、含んだ微量元素や紫外線照射などによって色が変化します。もともともっている特徴(微量元素など)と外からの要因(紫外線照射)によって色(気分、気持ち)が変わるのは、心と似ているのでは、と考えたのも「石」にした理由のひとつです。
- 自分の好みを多大に反映しているので、ミニアプリという形でこのようなアプリに需要があるのかをまず調査する目的もあります。
-- 需要があれば、アプリ化(気分トラッカー、鉱物収集機能付き)を検討したいです。
- パワーストーンを勧める意図はないので、結果画面の解説はあくまで石についてのものに終始しています。
